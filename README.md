# Expense Tracker (MERN Stack)

A full-stack **Expense Tracker** application built using the **MERN stack**.
This project helps users manage and analyze their daily expenses with an interactive dashboard and modern UI.

---

## 🚀 Features

* 🔐 **Authentication System**

  * Secure login and logout
  * Cookie-based authentication

* 💰 **Expense Management**

  * Add new expenses
  * Update existing expenses
  * Delete expenses
  * View all expenses in a structured table

* 📊 **Dashboard Analytics**

  * Summary cards for quick insights
  * Total and daily expense overview

* 📈 **Expense Charts**

  * Visual representation of expenses using charts

* 👤 **GitHub Style About Page**

  * Displays developer profile information
  * GitHub inspired dashboard layout

* ✉️ **Contact Form**

  * Simple contact section with user feedback notification

## ⚙️ Installation

### 1️⃣ Clone the repository

```bash
git clone https://github.com/yourusername/expense-tracker.git
```

### 2️⃣ Install dependencies

Frontend

```bash
cd frontend
npm install
```

Backend

```bash
cd backend
npm install
```

### 3️⃣ Run the application

Start backend:

```bash
npm run dev
```

Start frontend:

```bash
npm run dev
```
